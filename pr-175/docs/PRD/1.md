# Issue 1 — ASCIITennis: First-Person ASCII Tennis Game

## Features

- First-person view tennis game on TIC-80 with retro PS1-style 3D ASCII graphics
- Player controls character position with direction buttons
- Action button hits the tennis ball with multiple hit types
- AI opponent with selectable difficulty levels
- Rival selection at game start
- Standard tennis scoring system (win/loss conditions)

## Acceptance Criteria

- [ ] Game initializes on TIC-80 with ASCII-rendered 3D tennis court
- [ ] Player moves left/right/forward/backward on court using direction buttons
- [ ] Player can hit the ball with the action button; different button combinations produce different shot types (lob, slice, top spin, flat)
- [ ] AI rival has at least 2 difficulty levels (easy, hard)
- [ ] Player selects rival difficulty before the match starts
- [ ] Standard tennis scoring is implemented (15, 30, 40, deuce, advantage, game)
- [ ] Match ends when a player wins a set (first to 6 games with 2-game lead)
- [ ] Game loop includes serve, rally, point resolution
- [ ] Points, games, and sets are displayed on screen
