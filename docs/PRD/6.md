# Issue 6 — Screen/UI Iteration (画面迭代)

## Motivation

The current game uses a first-person 3D perspective projection rendered with ASCII depth-shaded characters. Testing confirms this display is unreadable — humans cannot meaningfully parse the court, track the ball, or play the game. The perspective view combined with monochrome ASCII shading creates an indecipherable visual experience.

## Reference: Mario Tennis GBC

Early Mario Tennis games (Game Boy Color) used a **top-down (overhead) 2D view** of the tennis court. Key visual properties:
- Entire court visible at once — both players on screen
- Clear, distinct court markings (baselines, sidelines, net, service boxes)
- Simple player sprites with clear positioning
- Ball visible and easy to track across the court
- Score HUD overlaid clearly on screen
- Color contrast between court surface, lines, players, ball, and HUD

## Features

- Replace 3D perspective camera with a **2D top-down ASCII representation** of the tennis court
- Both players visible on screen simultaneously
- Court rendered as a clear, recognizable tennis court with ASCII art walls/lines
- Player characters distinct and positional
- Ball movement clearly visible across the court
- HUD with readable score display
- Visual feedback for serves, hits, points, and shot types
- Maintain ASCII art aesthetic (green-on-black terminal look or enhanced ASCII palette)

## Acceptance Criteria

- [ ] Top-down court view replaces first-person perspective
- [ ] Both human and AI players visible on screen at all times
- [ ] Court boundaries, net, and service boxes clearly legible
- [ ] Ball position trackable without confusion
- [ ] HUD shows current score, games, sets legibly
- [ ] State overlays readable: serve prompt, point result, game over
- [ ] Player position updates smoothly in real time
- [ ] Ball trajectory visible (bounce, arc, speed)
- [ ] Game remains playable end-to-end (menu → match → game over → restart)

## Non-goals

- No changes to game logic, physics, AI, scoring, or state machine
- No changes to input system
- No new gameplay features
- No conversion to non-ASCII rendering
